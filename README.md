# DevOps Homelab

This repo documents the homelab environment I’m building on a Lenovo ThinkCentre M920s running Proxmox VE.

The goal of this project is to get more hands-on experience with infrastructure, virtualization, Linux administration, Docker, CI/CD pipelines, networking, monitoring, and self-hosted services using tools commonly found in real DevOps environments.

I started this lab to move beyond tutorials and actually build, break, troubleshoot, and maintain my own infrastructure.

---

Current Hardware

* Lenovo ThinkCentre M920s
* 16GB DDR4 RAM
* SSD Storage
* Intel CPU with virtualization support
* Proxmox VE Hypervisor

---

Current Progress

* Installed Proxmox VE
* Enabled VT-x / VT-d in BIOS
* Configured Proxmox networking
* Fixed Proxmox repository configuration issues
* Updated packages and configured no-subscription repositories
* Accessed Proxmox web UI successfully
